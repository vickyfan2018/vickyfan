HOLA

This is Coco Capitán’s personal portfolio website. As the user scrolls, images move into view from seemingly random points and then move back out of view as the user continues to scroll down. This kind of randomness breaks the grid system and creates a feeling that there are many photos being shown, which can attract the users to explore the website further. One can assume these images make up her portfolio, although they have neither a name or description associated with them.

Ben Mingo Folio

This is a website of an interactive and graphic designer called Ben Mingo. Users need to toggle the bar to open the webpage, and there are instruction and a circle to hint this interaction. This interactive element attracts users' attention and gives a good first impression. Moreover, the opening animation and transition is extremely impressive, the toggle bar spans and transforms from a horizontal rectanigle to a vertical rectangle, showing the work of the designer. The interaction is carefully designed too. When users click and toggle on the photo, the webpage becomes a naviagation setting. When user hovers on the title, the text becomes outlined, and hints the clickability of the text.

The Twelve Labours of Hercules

This is a website showcasing a collaboration of artists, which mainly foucuses on web graphics of creating hand's illustration. While laoding the screen, there are messages appearing on the page, which can lower the impatient feeling of user. The landing page requires users to click and toggle, then, the page will be divided into two horizontally, opening the index page. Besides, the 3D graphics of the hands are refined and realistics, creating an authentic experience on web.

Blacknegative

Blacknegative's website uses unusal horizontal navigation. To cue the user for the navigation system, there are symbol of hands with left and right arrows when users hover on the webpage, which gives users an inituitve start to toggle right to the left. The navigation menu appears when users hover on the black navigation bar at the bottom. When users toggle the bar, the small icon of each page slide in from the top right corner, which is a pleasant transition and animtaion. However, this webpage is not responsive, it would be nice to see how these elememts apply on mobile version.

Cavalier Conqueror of Excellence

This is a game which goal is to collect as many golden nuggets as possbile within 40 seconds. The instruction on the loading screen is user-friendly, as it reminds users to wear headphone and use chrome to experience the best web interaction. The opening animation of the types and logo is elegant, which presents a sophisticated and well-made quality. Although the dust in the game may be due to the poor quality of the 3D graphic, it successfully creates a mysteric atmosphere, which suits the feeling of the game.
